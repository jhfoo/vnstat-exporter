# vnstat-exporter
Prometheus exporter for vnstat

## Requirements
1. FreeBSD 13.1-RELEASE (tested).
2. vnstat installed, configured, and running.
3. zsh as default shell (tested).

## Install
```
git clone https://github.com/jhfoo/vnstat-exporter.git
cd vnstat-exporter
./bin/install
```

## Run
```
cd vnstat-exporter
./bin/start
```
